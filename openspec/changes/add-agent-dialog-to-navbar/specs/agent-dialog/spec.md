## ADDED Requirements

### Requirement: 智能体对话框显示在导航栏底部
系统SHALL在前端主界面导航栏（Layout.vue）的最底部显示一个智能体对话框组件。

#### Scenario: 页面加载时显示智能体对话框入口
- **WHEN** 用户登录系统并进入主界面
- **THEN** 导航栏底部显示智能体对话框入口按钮

### Requirement: 用户可以输入消息
系统SHALL允许用户在智能体对话框中输入文本消息。

#### Scenario: 用户输入文本消息
- **WHEN** 用户在输入框中输入文本
- **THEN** 输入框显示用户输入的内容

#### Scenario: 空消息不发送
- **WHEN** 用户输入框为空时点击发送按钮
- **THEN** 系统不发送消息

### Requirement: 对话框展开/收起功能
系统SHALL提供对话框展开和收起的切换功能。

#### Scenario: 点击按钮展开对话框
- **WHEN** 用户点击智能体对话框入口按钮
- **THEN** 对话框展开显示输入框和发送按钮

#### Scenario: 点击按钮收起对话框
- **WHEN** 用户点击已展开的对话框关闭按钮
- **THEN** 对话框收起只显示入口按钮
